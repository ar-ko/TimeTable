# Расписание ТюмГУ
Приложения для просмотра расписания Тобольского педагогического института им. Д.И. Менделеева.

![](demo.gif)

## О приложении
Представляет [расписание вуза](https://github.com/[username]/[repo_name]/issues/new) в удобоваримой форме.

## Использовалось:
* Xcode 11.3.*
* Swift 5.*

## Установка:
1. Скачать
2. Добавить ```ApiKey```
3. Пользоваться
